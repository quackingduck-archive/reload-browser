Reload-browser

A command line utility to reload the active browser tab

View the Project on GitHub quackingduck/reload-browser

  • Download ZIP File
  • Download TAR Ball
  • View On GitHub

A programmatic way to press the refresh button (or hit f5). From the terminal just run:

$ reload-browser

and the foremost tab/window will reload. To reload every time a file changes combine with a file watcher like wach:

$ wach "make index.html && reload-browser"
