reload-browser

A command line utility to reload the active browser tab

View the Project on GitHub

Install the browser extension then run:

$ reload-browser

and the foremost tab/window will reload.

To reload every time a file changes combine with a file watcher like wach:

$ wach "make index.html && reload-browser"

You can also reload just the css:

$ wach "make styles.css && reload-browser css"
